1 · Linear Equations

Exercise 1.1
Solve for x: 3x − 7 = 14
Solution
  1. Add 7 to both sides: 3x = 21
  2. Divide both sides by 3: x = 7
Answer: x = 7
Exercise 1.2
Solve for x: 2(x + 4) = 3x − 1
Solution
  1. Distribute: 2x + 8 = 3x − 1
  2. Subtract 2x from both sides: 8 = x − 1
  3. Add 1 to both sides: x = 9
Answer: x = 9
Exercise 1.3
Find the slope and y-intercept of: 4x − 2y = 8
Solution
  1. Solve for y: subtract 4x → −2y = −4x + 8
  2. Divide by −2: y = 2x − 4
  3. Slope m = 2, y-intercept b = −4
Answer: slope = 2, y-intercept = −4

2 · Quadratic Equations

Exercise 2.1 – Factoring
Solve by factoring: x² + 5x + 6 = 0
Solution
  1. Find two numbers that multiply to 6 and add to 5: 2 and 3
  2. Factor: (x + 2)(x + 3) = 0
  3. Set each factor to zero: x + 2 = 0 or x + 3 = 0
Answer: x = −2 or x = −3
Exercise 2.2 – Quadratic Formula
Solve using the quadratic formula: 2x² − 4x − 6 = 0
Solution
  1. Identify a = 2, b = −4, c = −6
  2. Compute discriminant: Δ = b² − 4ac = 16 + 48 = 64
  3. Apply formula: x = (4 ± 8) / 4
  4. Solutions: x = 3 or x = −1
Answer: x = 3 or x = −1
Exercise 2.3 – Completing the Square
Solve by completing the square: x² + 6x − 7 = 0
Solution
  1. Move constant: x² + 6x = 7
  2. Add (6/2)² = 9 to both sides: x² + 6x + 9 = 16
  3. Factor left side: (x + 3)² = 16
  4. Take square root: x + 3 = ±4
Answer: x = 1 or x = −7

3 · Functions

Exercise 3.1 – Domain & Range
Find the domain of: f(x) = √(x − 3)
Solution
  1. The expression under the radical must be non-negative.
  2. Set x − 3 ≥ 0
  3. Solve: x ≥ 3
Answer: Domain = [3, ∞)
Exercise 3.2 – Function Composition
Given f(x) = 2x + 1 and g(x) = x², find (f ∘ g)(3).
Solution
  1. First evaluate g(3) = 3² = 9
  2. Then evaluate f(g(3)) = f(9) = 2(9) + 1 = 19
Answer: (f ∘ g)(3) = 19
Exercise 3.3 – Inverse Functions
Find the inverse of: f(x) = 3x − 5
Solution
  1. Replace f(x) with y: y = 3x − 5
  2. Swap x and y: x = 3y − 5
  3. Solve for y: y = (x + 5) / 3
Answer: f⁻¹(x) = (x + 5) / 3

4 · Exponential & Logarithmic Functions

Exercise 4.1 – Exponential Equation
Solve: 2^(x+1) = 16
Solution
  1. Write 16 as a power of 2: 16 = 2⁴
  2. Set exponents equal: x + 1 = 4
  3. Solve: x = 3
Answer: x = 3
Exercise 4.2 – Logarithmic Equation
Solve: log₂(x − 1) = 4
Solution
  1. Rewrite in exponential form: x − 1 = 2⁴ = 16
  2. Solve: x = 17
Answer: x = 17
Exercise 4.3 – Natural Logarithm
Solve: e^(2x) = 5
Solution
  1. Take the natural log of both sides: ln(e^(2x)) = ln(5)
  2. Simplify left side: 2x = ln(5)
  3. Divide by 2: x = ln(5) / 2 ≈ 0.805
Answer: x = ln(5) / 2 ≈ 0.805

5 · Systems of Equations

Exercise 5.1 – Substitution Method
Solve the system:
y = 2x + 1
3x + y = 16
Solution
  1. Substitute y = 2x + 1 into the second equation: 3x + (2x + 1) = 16
  2. Simplify: 5x + 1 = 16 → 5x = 15 → x = 3
  3. Back-substitute: y = 2(3) + 1 = 7
Answer: x = 3, y = 7
Exercise 5.2 – Elimination Method
Solve the system:
2x + 3y = 12
4x − 3y = 6
Solution
  1. Add the two equations to eliminate y: 6x = 18
  2. Solve: x = 3
  3. Substitute into the first equation: 6 + 3y = 12 → 3y = 6 → y = 2
Answer: x = 3, y = 2
Exercise 5.3 – No Solution
Determine the number of solutions:
2x + y = 4
4x + 2y = 10
Solution
  1. Multiply the first equation by 2: 4x + 2y = 8
  2. Compare with the second equation: 4x + 2y = 10
  3. Left sides are identical but right sides differ (8 ≠ 10) — contradiction.
Answer: No solution (inconsistent system)
